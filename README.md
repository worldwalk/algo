# 算法学习项目

这是一个用于学习和练习算法的Go&Python语言项目，包含多种常见算法的实现和测试。

## 项目结构

```
algo/
├── cache/         # 缓存相关算法
├── dynprog/       # 动态规划算法
├── lcs/           # 最长公共子序列
├── lis/           # 最长递增子序列
├── list/          # 链表相关算法
├── maze/          # 迷宫问题
├── py/            # Python实现的算法
├── search/        # 搜索算法
├── sort/          # 排序算法
├── tree/          # 树结构算法
└── util/          # 工具函数
```

## 运行测试

### 通用测试命令
```bash
# 运行所有测试
go test ./...

# 运行特定包的测试
go test ./list

# 运行特定测试函数
go test ./list -run ^Test_reverseKGroup$ -v
```

### 注意事项
- 某些IDE或工具可能会自动添加`-test.fullpath`参数，但该参数在Go 1.20.4及更早版本中不存在
- 如果遇到`flag provided but not defined: -test.fullpath`错误，请使用手动构建的测试命令
- 推荐使用Go 1.21+版本以获得最新的测试功能支持
